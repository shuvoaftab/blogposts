## Step-by-Step Guide: Setting Up Pretty Links

1. Install and activate the plugin.
2. Navigate to **Pretty Links > Add New**.
3. Enter your destination URL and choose a custom slug.
4. Add a title and group (optional).
5. Save and insert the link into your content.
6. Track clicks from the **Reports** tab.

### Example:

```plaintext
Target URL: https://example.com/product/awesome-tool
Pretty Link: https://yourdomain.com/go/awesome-tool
```
